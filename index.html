A free simulation environment for artificial intelligence in games

WHY PALAIS?

GAME AI FOCUS
PALAIS is built for development and research of artificial intelligence in games. It provides a complete simulation environment with commonly required features such as time-dependant simulation, 3D visualisation and online knowledge inspection.
ACCESSIBILITY
PALAIS provides a simple, hassle-free JavaScript interface to access its feature set: No complicated configuration, compilation or deployment steps necesary. You can fully define the rules of your game as well as any behaviors directly from JavaScript code!
EXTENSIBILITY
PALAIS allows users to extend its core functionality by writing native code plugins. This way you are able add your own specialised, high-performance code to the simulation environment and even expose it to the scripting system if your use case requires it.
